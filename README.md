# エントランス屋さん太郎

DJイベントのエントランス業務をスマホから管理するWebアプリです。  
Firebase Realtime Database でリアルタイム同期し、複数人で同じイベントを同時に操作できます。

---

## 使い方

### 1. イベントを準備する

アプリを開くと自動でイベントIDが発行されます。  
⚙️ → **イベント設定** からイベント名と好みのIDを変更できます。

> イベントIDは URL の `?event=xxx` 部分です。同じIDを開けば同じデータに接続されます。

---

### 2. 演者リストを登録する

**方法A：タイムテーブル画像から読み込む**

⚙️ → **タイムテーブル読込** → 画像を選択 → 「画像から読み取り」

以下のフォーマットの画像に対応しています。

```
18:00-18:30 saki          ← 時間が先頭
saki 18:00-18:30          ← 名前が先頭
18:00 saki                ← 終了時間なしもOK
```

読み取り結果を確認・修正してから「演者に反映」で保存。

**方法B：テキストで直接入力する**

⚙️ → **演者を手動追加**、または演者カードを長押しで編集ダイアログが開きます。

一括入力したい場合は ⚙️ → **タイムテーブル読込** のテキストエリアに直接貼り付けも可能です。

```
18:00,saki,1
18:30,Dagobah,1
20:00,ミコヤン,2,もう1人後入り   ← メモ付き
```

フォーマット：`時間,演者名,人数,メモ`（時間・メモは省略可）

---

### 3. 入場料を設定する

⚙️ → **入場料を追加・編集**

| 項目 | 説明 |
|------|------|
| 料金名 | 通常 / ディスカウント / ゲスト など |
| 金額 | 500円刻みで入力 |
| D（ドリンク枚数） | ドリンクチケットの枚数 |

---

### 4. 当日の操作

#### 入場料カウント

トップ画面の入場料カードの **＋ / −** ボタンでカウントします。  
合計金額・ドリンク枚数・人数がリアルタイムで集計されます。

長押しで入場料設定ダイアログが開きます。

#### 演者チェック

演者カードの **＋ / −** ボタンで到着人数を記録します。  
全員到着すると名前が緑色に変わります。

- **全員タブ**：タイムテーブル順に表示
- **未到着タブ**：まだ揃っていない演者だけ表示
- **検索ボックス**：名前・メモで絞り込み

演者カードを**長押し**すると個別編集ダイアログが開きます（名前・時間・人数・到着数・メモの修正、削除も可能）。

#### 現在・次の演者表示

ヘッダーに現在演奏中と次の演者が自動表示されます（10秒ごとに更新）。  
深夜0〜5時台のイベントにも対応しています。

#### 連絡メモ

「遠隔爆撃・連絡メモ」欄はフロアとエントランス間の伝言に使えます。  
保存するとFirebase経由でリアルタイムに同期されます。

#### ライブCH指定・備考

機材チャンネル指定やフード・物販の配置メモなど、当日スタッフ共有情報を記入できます。

---

### 5. URLを共有する

「共有URL」ボタンでクリップボードにコピーされます。  
同じURLを開いたスタッフ全員がリアルタイムで同期されます。

---

## 設定メニュー（⚙️）一覧

| メニュー | 内容 |
|----------|------|
| イベント設定 | イベント名・IDの変更 |
| タイムテーブル読込 | 画像OCRまたはテキストで演者登録 |
| 演者を手動追加 | 1人ずつ追加 |
| 入場料を追加・編集 | 料金種別の管理 |
| 到着状況だけリセット | 演者リストを残して全員0に戻す |
| 入場料カウントだけリセット | 料金設定を残してカウントを0に戻す |
| イベントを複製 | 演者・タイムテーブルを新IDにコピー（到着・カウントはリセット） |
| イベント全削除 | データを完全削除（取り消し不可） |

## ファイル構成

```
index.html        アプリ本体（単一ファイル）
manifest.json     PWA設定
icon-180.png      ホーム画面アイコン（iOS）
icon-192.png      ホーム画面アイコン（Android）
icon-512.png      ホーム画面アイコン（大）
```

---

## 動作環境

- モダンブラウザ（Chrome / Safari / Firefox）
- スマートフォン推奨（iOS Safari / Android Chrome）
- PWA対応：ホーム画面に追加するとアプリとして起動できます
- インターネット接続が必要です（Firebase使用）

---

## 技術スタック

- **Firebase Realtime Database**：リアルタイム同期
- **Tesseract.js**：タイムテーブル画像のOCR読み取り
- フレームワークなし・単一HTMLファイル
